目前列印模式

偵測中

系統會優先把小票送到網路印表機。沒有設定印表機 IP 時，會在伺服器的 data/print-jobs 產生備用文字小票，方便人工補印。

紙張80 x 80mm 熱感小票
連線USB + 網口機型建議固定 IP
後端連接埠9100 ESC/POS

加盟店配置方式

建議
  1. 印表機接到店內路由器，設定固定 IP，例如 192.168.1.88。
  2. 後端啟動時設定 PRINTER_HOST=印表機IP，PRINTER_PORT 預設 9100。
  3. 每張新單可在接單工作台直接列印，查單系統也可補印。
